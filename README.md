## 🌟 Project Overview

**The Pearl Awaits** is a visual-first web concept that showcases Uganda’s landscapes, heritage, and travel opportunities. The goal is to deliver an **immersive, interactive experience** for users exploring the country online.



## 🎨 Design Philosophy

* **Modern UI:** Bento-style layouts, Glassmorphism, and Parallax scrolling.
* **Cultural Spotlight:** Sections highlighting the Buganda Kingdom, Karamojong Manyattas, and the Batwa Forest People.
* **UX-Centered:** Optimized for performance, mobile responsiveness, and intuitive navigation.



## 🛠️ Tech Stack

* **Frontend:** React.js / Next.js
* **Styling:** Tailwind CSS (custom dark theme)
* **Animations:** Framer Motion
* **Visuals:** AI-generated high-fidelity cultural and landscape imagery



## ✨ Key Features

* **Interactive Itineraries:** Reimagined travel routes such as the Entebbe Loop and safari circuits.
* **Heritage Cards:** Visually rich cultural storytelling.
* **Traveler Toolkit:** Real-time utilities for international tourists.

## ⚙️ Installation

To run this project locally:

```bash
# Clone the repository
git clone <repo-url>

# Navigate into the project
cd pearl-awaits

# Install dependencies
npm install

# Start development server
npm run dev
```

Open your browser and navigate to `http://localhost:3000`.
